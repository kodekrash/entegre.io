Navigation bars

Class name

E.bootstrap.navbar (0.2.0)

Methods

Method Parameters Returns Description Example
attr key, value this Add attribute key/value to component attr('id','mynav')
build string Build string of component markup build()
append jquery selector boolean Build and append string to DOM append('#mydiv')
brand any buildable this Set brand content brand('My Site')
item url, label, [active], [disabled] this Add menu item item('#','Action')
inverse this Set inverse mode (.navbar-inverse) inverse()

Examples

Default
Brand
  • Item 1 (normal)
  • Item 2 (active)
  • Item 3 (disabled)
  • Item 4
$E('navbar')
  .brand(E.node('a',{'href':'#','class':'navbar-brand'},'Brand'))
  .item('#','Item 1 (normal)')
  .item('#','Item 2 (active)',true)
  .item('#','Item 3 (disabled)',false,true)
  .item('#','Item 4');
