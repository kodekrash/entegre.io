Tabs & Pills

Class name

E.bootstrap.tabs (0.2.0)

Methods

Method Parameters Returns Description Example
attr key, value this Add attribute key/value to component attr('id','myaccordion')
build string Build string of component markup build()
append jquery selector boolean Build and append string to DOM append('#mydiv')
card title, body [, attr] this Add card to content deck card('Tab 1','Tab 1 content')
pills this Set pill mode (.nav-pills) pills()
justify this Set justified layout mode (.nav-justified) justify()
