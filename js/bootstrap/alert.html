Alerts

Class name

E.bootstrap.alert (0.2.0)

Methods

Method Parameters Returns Description Example
child any buildable this Add content to component child('Hello world')
attr key, value this Add attribute key/value to component attr('id','myalert')
build string Build string of component markup build()
append jquery selector boolean Build and append string to DOM append('#mydiv')
mode valid mode
except default, primary		 this Set mode (.alert-{{mode}}) mode('success')
callout any buildable this Set emphasis text. Call before child if used. callout('Success message')

Examples

Success with callout
Success Alert message text
$E('alert')
  .mode('success')
  .callout('Success')
  .child('Alert message text');
<div class="alert alert-success"><strong>Success</strong> Alert message text</div>
Success without callout
Alert message text
$E('alert')
  .mode('success')
  .child('Alert message text');
<div class="alert alert-success">Alert message text</div>
